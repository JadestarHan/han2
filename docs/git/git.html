git tag的新建和删除

  • 创建tag((本地)) git tag -a v1.0 -m "msg"
  • 查看tag git tag
    • 要显示附注信息使用show指令git show v1.0
  • 创建tag(远程)git push origin --tags
  • 删除tag(本地)git tag -d v1.0
  • 删除tag(远程) git push origin :refs/tags/v1.0
  • 获取远程版本 git fetch origin tag v1.0
